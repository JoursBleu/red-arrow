Quick Start

快速开始

  1. Install安装
    Download APK from GitHub Releases 从 GitHub Releases 下载 APK
  2. Configure配置
    Enter SSH server host, port, username, and choose auth method 填写 SSH 服务器地址、端口、用户名，选择认证方式
  3. Connect连接
    Tap Connect — SOCKS5 and HTTP proxies start instantly 点击连接 — SOCKS5 和 HTTP 代理立即启动
  4. Use使用
    Set your phone or LAN devices to use the proxy address 在手机或局域网设备上设置代理地址即可使用
